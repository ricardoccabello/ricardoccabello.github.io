IV Production

Five things I check before sending a project to production

First, dimensions. Not only the overall size, but the clearances, thicknesses and interfaces with the existing space.

Second, materials. The finish should be identified clearly enough that purchasing and the workshop understand the same thing.

Third, assembly. I check how each part arrives, how it is joined and whether there is enough room to build it on site.

Fourth, responsibility. Every item needs an owner. Unassigned details are usually the ones that fail.

Finally, consistency. The plan, elevations, schedules and written notes must all describe the same project.
