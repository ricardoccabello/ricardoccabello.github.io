II Reuse

Why existing furniture can improve a project

Existing furniture is often treated as a restriction. I prefer to see it as part of the brief.

Reusing an element can reduce cost, transport and installation time, but it only works when the new layout gives that element a clear role.

The useful question is not whether an object is old or new. It is whether it still supports the space, the brand and the way people will use it.
