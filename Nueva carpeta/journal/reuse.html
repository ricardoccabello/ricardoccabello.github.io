II Reuse

When existing furniture deserves another layout

Existing furniture is often introduced as a restriction. I prefer to treat it as one more condition of the brief—something to assess rather than automatically remove.

Reuse can reduce cost, transport and lead time, but keeping an element only makes sense when the new layout gives it a clear purpose and the piece still meets the project requirements.

The useful question is not whether something is old or new. It is whether it still supports the space, the brand and the way people need to use it.
